## section6.3 SearchPresenter Component

```javascript
handleSubmit = (event) => {
    event.preventDefault();
    const { searchTerm } = this.state;
    if(searchTerm !== ""){
        this.searchByTerm();
    }
};

updateTerm = (event) => {
    const { 
        target: { value } 
    } = event;
    console.log(value);
    this.setState({
        searchTerm: value
    });
};
```

```event.preventDefault();```는 event를 취소할 수 있는 경우, 검색어를 입력후 enter를 누르면 발생하는 event의 전파를 막지 않고 그 이벤트를 취소합니다.
